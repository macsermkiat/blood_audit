## กฎ classification

`INSUFFICIENT_EVIDENCE` ต้องอยู่เป็น class แยก เพราะหมายถึงหลักฐานไม่พอ
ไม่ใช่หลักฐานว่าการให้เลือดไม่เหมาะสม การรวมค่านี้เข้า `INAPPROPRIATE`
จะทำให้รายงาน QI บิดความหมาย

## Reproducibility fields

รายงานควรมี hash ของ policy, model, redactor, prompt และ evidence bundle
เพื่อให้ตรวจย้อนหลังได้ว่าผลลัพธ์เกิดจากข้อมูลและกฎชุดใด
