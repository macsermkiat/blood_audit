## blood_audit ทำอะไร

blood_audit อ่านข้อมูลส่งออกจาก HOSxP ที่เสร็จสมบูรณ์แล้ว
ตรวจสอบการให้เลือดแดงเข้มข้น (RBC, red blood cell) แต่ละครั้ง
เทียบกับนโยบายฮีโมโกลบิน (Hb) แบบ 3-ชั้น แล้วเขียนผลออดิท 1 แถว
ต่อหน่วยเลือด 1 หน่วย

การให้เลือดแต่ละครั้งจะได้รับ 1 ใน 3 ป้าย:

- `APPROPRIATE` — Hb ต่ำกว่าเกณฑ์ของกลุ่ม หรือเข้าเงื่อนไข
  acute-blood-loss override
- `POTENTIALLY_INAPPROPRIATE` — Hb สูงกว่าเกณฑ์และไม่มีเงื่อนไข override
- `LLM_REVIEW` — Hb สูงกว่า 10 g/dL ส่งไปให้ large language model (LLM)
  วิเคราะห์เหตุผลทางคลินิก โดยมีการตรวจสอบทุก quote กับข้อมูลต้นทาง
