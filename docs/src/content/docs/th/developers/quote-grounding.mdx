## Fail-closed หมายถึงอะไร

ถ้า mandatory layer ใดตอบ false ระบบจะเก็บ
`final_classification = NEEDS_REVIEW` และ `review_reason =
"hallucination_suspect"` แถวนั้นจะไม่ถูก downgrade, retry ด้วยกฎที่อ่อนลง
หรือยอมรับเป็น `APPROPRIATE` / `INAPPROPRIATE` แบบเงียบ ๆ

`hallucination_suspect` เป็นค่าใน `review_reason` ไม่ใช่ classification
จริง ค่า classification ที่เก็บได้คือ `APPROPRIATE`, `INAPPROPRIATE`,
`NEEDS_REVIEW`, `INSUFFICIENT_EVIDENCE` และ `POTENTIALLY_INAPPROPRIATE`

## Medical-NLI เป็น optional

`bba.quote_grounder.verify_citation(..., nli_gate=None)` ไม่รัน NLI โดย default
จะประเมินชั้น medical-NLI เฉพาะเมื่อ caller ส่ง `NLIEntailmentGate` เข้ามา

## สิ่งที่ห้ามทำ

- ห้าม mock หรือ short-circuit grounding layer ใน test
- ห้ามเพิ่ม flag เพื่อ skip grounding
- ห้ามลด minimum quote length 25 characters
- ห้ามแทน medical-NLI ด้วย string check ธรรมดา
