## คำสั่งหลัก

```bash
uv run ruff check .
uv run ruff format --check .
uv run mypy src
uv run pytest
```

`mypy tests` เป็น advisory ในตอนนี้ เพราะยังมี debt ใน test helpers
แต่ `mypy src` ต้องผ่านแบบ strict

## LLM และ cost guard

test ที่เกี่ยวกับ LLM ต้องใช้ cassette หรือ fixture เท่านั้น ห้ามเรียก live API
โดยไม่ตั้งใจ cost guard ใน `bba.audit_pipeline.cost_guard` จะ block
live access ใน test context
