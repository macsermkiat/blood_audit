## What blood_audit does

blood_audit reads completed HOSxP exports from the hospital information
system. It checks each red blood cell (RBC) transfusion against the
3-tier hemoglobin (Hb) policy. It writes one audit row per transfusion.

Each transfusion gets one of three labels:

- `APPROPRIATE` — Hb is below the cohort threshold, or the acute-blood-loss
  override applies.
- `POTENTIALLY_INAPPROPRIATE` — Hb is above the threshold and no override
  conditions are documented.
- `LLM_REVIEW` — Hb is above 10 g/dL. The row is sent to a large language
  model (LLM) for clinical reasoning, with every quote checked against
  the source evidence.

Six fail-closed quote-grounding layers protect every LLM_REVIEW row from
hallucination. The deterministic classifier remains the authoritative source.
