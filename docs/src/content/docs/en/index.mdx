## What blood_audit does

Reads finished HOSxP exports, classifies each RBC transfusion against a
**3-tier hemoglobin policy** with cohort-aware thresholds and an
acute-blood-loss override, and emits an audit row per transfusion:

- `APPROPRIATE` — Hb below the cohort threshold (or acute-blood-loss override fires).
- `POTENTIALLY_INAPPROPRIATE` — Hb above threshold and no override conditions.
- `LLM_REVIEW` — Hb above 10 g/dL, routed to an LLM for quote-grounded clinical reasoning.

Six fail-closed quote-grounding layers protect every LLM_REVIEW row from
hallucination. The deterministic classifier remains the authoritative source.
