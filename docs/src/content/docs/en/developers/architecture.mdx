## Invariants worth keeping

1. **Deterministic-first.** The deterministic classifier is authoritative.
   The LLM is only consulted for Tier 3 (Hb > 10 g/dL).
2. **Idempotent re-runs.** `(run_id, encounter_id)` is unique in the store;
   re-runs no-op unless `--force` is passed.
3. **Quote-grounded LLM output.** All six grounding layers run; failure
   routes to `hallucination_suspect`, never to a result row.
4. **No real-time path.** No subscriber to the audit stream emits clinician
   alerts. This is QI, not SaMD.
5. **De-identification happens before any cross-container hop.** The
   `bba.deid_redactor` step is non-optional.

## Where the integration seams live

`bba.dashboard`, `bba.report_generator`, `bba.eval_harness`, and the
sentinel orchestration in `bba.monitoring` ship complete but are not wired
to the CLI by default. See [Integration seams](/blood_audit/en/operators/integration-seams/).
